基于 HarmonyOS ArkUI 框架开发的经典贪吃蛇游戏，使用 Canvas 绘制，支持触摸/鼠标方向控制，计分和重新开始功能。

📱 效果预览
20×20 游戏网格，每个格子 20px

蛇身绿色，食物红色，网格线浅灰

顶部显示当前得分和游戏状态

底部四个方向按钮控制蛇的移动

🎮 游戏规则
使用方向按钮（↑ ↓ ← →）控制蛇的移动方向。

吃到红色食物可以增加长度和得分。

撞到墙壁或自己的身体则游戏结束。

点击“新游戏”按钮可以重新开始。

🛠️ 技术栈
开发工具：DevEco Studio 5.0+

编程语言：ArkTS

UI 框架：HarmonyOS ArkUI

绘图：Canvas API

🚀 如何运行
安装 DevEco Studio 并配置好 HarmonyOS SDK。

克隆或下载本项目代码。

用 DevEco Studio 打开项目，等待 Gradle 同步完成。

连接真机或启动模拟器（建议使用 Phone 类型，API 10+）。

点击工具栏的 Run 按钮（或按 Shift+F10）编译运行。

📝 自定义配置
你可以在 Index.ets 顶部修改以下常量来调整游戏：

变量名	说明	默认值
gridSize	网格大小（格数）	20
cellSize	每个格子像素大小	20
speedMs	蛇移动间隔（毫秒）	150
initSnake	初始蛇身坐标数组	见代码
🧪 已知问题
如果模拟器上方向按钮过大，可以调整按钮的 width/height 属性。

真机调试时请确保开启“允许访问 USB 设备”和“显示悬浮窗”权限（若需要）。
